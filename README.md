# TURKISH PANEL 🇹🇷

Веб-панель для автоматизации действий Facebook (SMM Hybrid Panel).

## Деплой на Windows Server (Дедик)

### 1. Подготовка системы
Убедитесь, что установлен **Python 3.11+**. При установке обязательно поставьте галочку **"Add Python to PATH"**.

### 2. Подготовка проекта
Распакуйте архив `turkish_panel.zip` в удобную папку (например, `C:\turkish_panel\`).

### 3. Установка виртуального окружения и зависимостей
Откройте **PowerShell** или **Командную строку (cmd)** от имени Администратора, перейдите в папку с проектом и выполните:

```powershell
cd C:\turkish_panel\

# Создаем виртуальное окружение
python -m venv .venv

# Активируем его
.\.venv\Scripts\activate

# Устанавливаем зависимости
pip install uv
uv pip install -r pyproject.toml

# Устанавливаем браузеры для Playwright
playwright install chromium
playwright install-deps chromium
```

### 4. Настройка окружения (.env)
Отредактируйте файл `.env` (через Блокнот или Notepad++) и укажите свои ключи (MoreThanPanel, Apify) и доступы администратора.

### 5. Запуск панели
Находясь в активированном виртуальном окружении (`(.venv)` в начале строки), запустите сервер:

```powershell
fastapi run api.py --port 8000 --host 0.0.0.0
```

> **Важно:** Не закрывайте это черное окно консоли, пока панель должна работать. Фоновый воркер работает именно в нем.

Панель будет доступна по адресу `http://IP_ВАШЕГО_СЕРВЕРА:8000/`.

### 6. Загрузка аккаунтов
Откройте панель в браузере, перейдите в раздел "Аккаунты", нажмите **"ЗАГРУЗИТЬ .TXT"** и выделите свои файлы аккаунтов. Они загрузятся автоматически.

**Важно для аккаунтов без кук:**
Если вы загрузили аккаунты в формате `facebook giriş / şifre`, у них не будет кук. В таблице аккаунтов:
1. Выберите **Прокси** для такого аккаунта.
2. Нажмите на иконку **Ключа** (Проверить вход).
3. Бот зайдет по логину и паролю через выбранный прокси, получит куки и сохранит их.
4. После этого аккаунт готов к выполнению задач.
